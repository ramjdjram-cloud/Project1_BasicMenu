# Basic Menu UI

A simple and clean navigation menu built with **HTML** and **CSS**.

This project is intended as a beginner-friendly layout or boilerplate for building a basic website header. It includes:

- ✅ A responsive **navbar**
- 🎯 A **logo** image
- 🏷️ A **title**
- 🔗 Three navigation **links**
